def create_sequences(data, time_steps=60):
    X = []
    y = []

    for i in range(time_steps, len(data)):
        X.append(data[i - time_steps:i, 0])
        y.append(data[i, 0])

    return np.array(X), np.array(y)

model = Sequential([
    LSTM(
        units=64,
        return_sequences=True,
        input_shape=(time_steps, 1)
    ),

    Dropout(0.2),

    LSTM(
        units=32,
        return_sequences=False
    ),

    Dropout(0.2),

    Dense(units=16, activation="relu"),

    Dense(units=1)
])

model.compile(
    optimizer="adam",
    loss="mean_squared_error"
)

future_predictions_scaled = []

    input_sequence = last_sequence.reshape(
        1,
        time_steps,
        1
    )

    next_prediction = model.predict(
        input_sequence,
        verbose=0
    )[0, 0]

    future_predictions_scaled.append(
        next_prediction
    )

    last_sequence = np.append(
        last_sequence[1:],
        [[next_prediction]],
        axis=0
    )

✅ Conclusion¶

In this notebook, we successfully implemented an LSTM neural network for Apple stock price forecasting.

The model was trained using historical closing prices and evaluated using:

  • Mean Absolute Error
  • Root Mean Squared Error
  • Mean Absolute Percentage Error
  • Actual vs predicted forecast visualization

We also generated closing-price forecasts for the next 30 business days.

LSTM is useful for time-series forecasting because it can learn long-term dependencies and sequential patterns from historical observations. However, real stock prices are also influenced by news, economic conditions and market sentiment, so the future predictions should be treated as experimental forecasts rather than financial advice.
